# MOTORWAY Fuel Station

Hyper-premium fuel station management system web app.

## Tech Stack
- React
- Tailwind CSS
- Lucide React Icons

## Getting Started

```bash
npm install
npm run dev
```

Then open the local Vite URL shown in the terminal.

## Vercel Deployment

- Framework: Vite
- Build Command: `npm run build`
- Output Directory: `dist`
